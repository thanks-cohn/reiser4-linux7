# Long Names Are The Soul

Reiser4-NX must not become a shittier version of existing filesystems.

One of the project’s defining ambitions is the restoration of huge, expressive, human file names: names long enough to preserve meaning, memory, context, and intent.

A filesystem should not force human thought into tiny machine labels forever.

Stability comes first.

Data safety comes first.

Recovery comes first.

But once the crown of stability is earned, Reiser4-NX must pursue the long-name dream seriously.

The target is simple:

Users should be able to give files absurdly long, expressive names approaching 4000 characters.

If Linux can safely expose those names natively, we pursue native large names.

If Linux cannot safely expose them through ordinary VFS paths yet, Reiser4-NX must still preserve them as true names: searchable, exportable, recoverable, and restorable.

The POSIX name is the passport.

The Reiser4-NX true name is the soul.

We obey Linux where we must.

We transcend Linux where meaning can live safely.

We do not abandon the long-name dream.
